Sistema Octal

El sistema octal es un sistema de numeración de base 8 que utiliza los dígitos del 0 al 7. Es especialmente útil en computación por su relación directa con el sistema binario.

¿Qué es el Sistema Octal?

El sistema octal, también conocido como sistema de base 8, utiliza únicamente los dígitos del 0 al 7 para representar cualquier número. Cada posición en un número octal representa una potencia de 8.

Conversión de Decimal a Octal

Para convertir un número decimal a octal, dividimos sucesivamente entre 8 y tomamos los residuos en orden inverso.

Ejemplo: Convertir 95 a octal

95 ÷ 8 = 11 residuo 7
11 ÷ 8 = 1 residuo 3
1 ÷ 8 = 0 residuo 1

Leyendo los residuos de abajo hacia arriba: 137
Por lo tanto, 95₁₀ = 137₈

Conversión de Octal a Decimal

Para convertir de octal a decimal, multiplicamos cada dígito por la potencia correspondiente de 8 y sumamos todos los resultados.

Ejemplo: Convertir 137₈ a decimal

137₈ = 1×8² + 3×8¹ + 7×8⁰
     = 1×64 + 3×8 + 7×1
     = 64 + 24 + 7
     = 95₁₀
