Sistema Hexadecimal

El sistema hexadecimal es un sistema de numeración de base 16 ampliamente utilizado en programación y ciencias de la computación por su compacta representación de valores binarios.

¿Qué es el Sistema Hexadecimal?

El sistema hexadecimal, también conocido como sistema de base 16, utiliza los dígitos del 0 al 9 y las letras de la A a la F para representar cualquier número. Cada posición en un número hexadecimal representa una potencia de 16.

Conversión de Decimal a Hexadecimal

Para convertir un número decimal a hexadecimal, dividimos sucesivamente entre 16 y tomamos los residuos en orden inverso.

Ejemplo: Convertir 255 a hexadecimal

255 ÷ 16 = 15 residuo 15 (F)
15 ÷ 16 = 0 residuo 15 (F)

Leyendo los residuos de abajo hacia arriba: FF
Por lo tanto, 255₁₀ = FF₁₆

Conversión de Hexadecimal a Decimal

Para convertir de hexadecimal a decimal, multiplicamos cada dígito por la potencia correspondiente de 16 y sumamos todos los resultados.

Ejemplo: Convertir A3₁₆ a decimal

A3₁₆ = A×16¹ + 3×16⁰
     = 10×16 + 3×1
     = 160 + 3
     = 163₁₀
