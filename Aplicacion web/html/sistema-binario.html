Sistema Binario

El sistema binario es un sistema de numeración en el que los números se representan utilizando solamente dos dígitos: 0 y 1. Es la base de toda la computación digital.

¿Qué es el Sistema Binario?

El sistema binario, también conocido como sistema de base 2, utiliza únicamente los dígitos 0 y 1 para representar cualquier número. Cada posición en un número binario representa una potencia de 2.

Conversión de Decimal a Binario

Para convertir un número decimal a binario, dividimos sucesivamente entre 2 y tomamos los residuos en orden inverso.

Ejemplo: Convertir 13 a binario

13 ÷ 2 = 6 residuo 1
6 ÷ 2 = 3 residuo 0
3 ÷ 2 = 1 residuo 1
1 ÷ 2 = 0 residuo 1

Leyendo los residuos de abajo hacia arriba: 1101
Por lo tanto, 13₁₀ = 1101₂

Conversión de Binario a Decimal

Para convertir de binario a decimal, multiplicamos cada dígito por la potencia correspondiente de 2 y sumamos todos los resultados.

Ejemplo: Convertir 1101₂ a decimal

1101₂ = 1×2³ + 1×2² + 0×2¹ + 1×2⁰
      = 1×8 + 1×4 + 0×2 + 1×1
      = 8 + 4 + 0 + 1
      = 13₁₀
