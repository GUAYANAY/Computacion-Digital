Leyes y Postulados del Álgebra de Boole

Las leyes y postulados permiten la manipulación algebraica de expresiones lógicas, siendo esenciales para la simplificación de circuitos.

Leyes Principales (con A, B y C como variables)

  • Leyes de Identidad: A + 0 = A y A * 1 = A
  • Leyes de Dominación (o Nulo): A + 1 = 1 y A * 0 = 0
  • Leyes de Idempotencia: A + A = A y A * A = A
  • Leyes de Complemento (o Inversión): A + (NOT A) = 1 y A * (NOT A) = 0
  • Doble Negación (o Involución): NOT (NOT A) = A
  • Leyes Conmutativas: A + B = B + A y A * B = B * A
  • Leyes Asociativas: (A + B) + C = A + (B + C) y (A * B) * C = A * (B * C)
  • Leyes Distributivas: A * (B + C) = (A * B) + (A * C) y A + (B * C) = (A + B) * (A + C)
  • Leyes de Absorción: A + (A * B) = A y A * (A + B) = A
  • Teoremas de De Morgan: NOT (A * B) = (NOT A) + (NOT B) y NOT (A + B) = (NOT A) * (NOT B)
Consejo: Memoriza estas leyes para simplificar expresiones de forma rápida y correcta.
