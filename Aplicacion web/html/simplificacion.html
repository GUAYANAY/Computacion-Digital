Simplificación de Expresiones Booleanas

La simplificación de expresiones es el proceso de reducir una expresión lógica a su forma más sencilla, lo que resulta en un circuito digital más eficiente y con menos componentes.

Herramienta de Simplificación (Simulación)

Ingresa tu expresión. Usa A-Z para variables, + para OR, · para AND (o yuxtaposición), y ' para NOT (negación).

Métodos de Simplificación

Existen dos métodos principales para simplificar funciones booleanas:

1. Simplificación Algebraica

Se utiliza aplicando directamente las Leyes de Boole para eliminar términos redundantes o agrupar variables.

2. Mapas de Karnaugh

Un método visual y sistemático para simplificar expresiones con hasta cuatro o cinco variables.

Ejemplo de Simplificación (Algebraica)

Simplificar la función: F = AB + AB'

  1. F = A(B + B') (Ley Distributiva)
  2. F = A · 1 (Ley de Complemento: B + B' = 1)
  3. F = A (Ley de Identidad: A · 1 = A)

La expresión final simplificada es F = A.
